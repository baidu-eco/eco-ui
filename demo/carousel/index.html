Carousel 实例

  1. view { width: undefined, height: undefined }Object

    轮播的可视区域高宽设置

  2. step: 'auto'String, Number

    步长根据 view 设置中的高或宽进行同步

  3. data: undefinedArray

    如果该参数没有设置内容，则直接从element 中根据 selector 获取

  4. selector { outer: 'ul', inner: 'li' }

    选择器

  5. template: { outer: '<${outer} ...>', inner: '<${inner}>{data}' },Object

    如果 data 为 Array 时，进行dom 渲染时的替换模板

  6. indexButton: trueboolean

    索引按钮，该按钮根据 轮播容器的总长度 / step 来输出按钮数量

  7. isPages: falseboolean

    是否显示页码，当前页面根据 每项 在可视范围内 的 个数 进行展示

  8. dirButton: trueboolean

    方向键按钮

  9. dirButtonAutoHide: trueboolean

    方向键按钮是否在鼠标未悬停的时候隐藏

  10. duration: 300Number

    动画时间

  11. index: 0Number

    默认展示第几项

  12. animateType: 'silde'String [ 'fade', 'silde' ]

    动画效果类型

  13. animateDir: 'hori'String [ 'hori', 'vert' ]

    如果是位移效果，可设置水平或垂直方向

  14. autoPlay: trueboolean

    是否自动播放

  15. autoDelay: 4000Number

    自动播放的间隔时间 (ms)

  16. update: function(elems) {}Function

    从外部更改你想改的...

  17. onAnimateBefore: function(runner) {}Function
  18. onAnimateComplete: function(runner) {}Function
